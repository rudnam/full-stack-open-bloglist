# bloglist

## Dev

Install dependencies

```bash
npm install
cd frontend/
npm install
```

Start backend

```bash
cd full-stack-open-bloglist/
npm run start
```

Run another terminal and start frontend

```bash
cd full-stack-open-bloglist/frontend/
npm run start
```
